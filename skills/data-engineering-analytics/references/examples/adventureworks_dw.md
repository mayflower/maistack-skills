# AdventureWorks DW Demo Notes

AdventureWorks DW is the current demo database for building and testing the analytics interface. These notes are optional context only. Executable scripts and dashboard generation must remain database-generic.

Typical demo analyses include sales trends, category contribution, product drilldown, geography segmentation, customer segmentation, and data-quality checks.

When the connected schema resembles AdventureWorks, likely business concepts include sales facts, product dimensions, customer dimensions, geography dimensions, date dimensions, and reseller/internet sales channels. Verify all table and column names from the live schema profile before querying.
